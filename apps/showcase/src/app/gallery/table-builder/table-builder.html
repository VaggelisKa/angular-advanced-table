Visually configure your table primitives, preview the grid interactively, and copy the clean Angular code.

Configure Features

Adds fuzzy-search inputs to query row content globally. Renders interactive visibility chips to toggle columns. Enables paging logic with page-size chips and pager navigation. Allows columns to be stuck to the left/right boundaries. Enables drag-and-drop, header-menu move actions, and Ctrl+Shift+Arrow keyboard column reordering (Command+Shift+Arrow on macOS). Renders a horizontal scrolling slider for wide tables. Pins the table header to the top of the viewport during scroll.
