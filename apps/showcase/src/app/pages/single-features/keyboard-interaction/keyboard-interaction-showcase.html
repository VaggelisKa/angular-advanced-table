Keyboard Interaction

Demonstrates the ARIA grid cell-interaction model: a cell whose only content is a single button focuses it directly, while cells with several controls use Enter to step in, Tab to move between the controls, and Escape to return focus to the cell.

Interactive Cells Grid

How To Use The Keyboard

  1. Tab to the grid, then move between cells with the Arrow keys.
  2. Move onto an Actions cell: its Acknowledge button is the cell's only content, so it is focused directly — press Enter or Space to activate it.
  3. Move onto a Status cell: it mixes text with a checkbox, so focus stays on the cell — press Enter to reach the checkbox, then Space to toggle it.
  4. Press Enter on a header to move focus into its controls (sort button and column menu).
  5. Press Tab / Shift + Tab to walk between the controls within the cell; past the last control, Tab leaves the grid.
  6. Press Escape to return focus from a header control to its cell.
