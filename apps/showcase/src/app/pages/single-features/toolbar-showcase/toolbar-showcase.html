Table Toolbar

Composable toolbar shell with static start/center/end slots and roving keyboard focus. Items pick their slot with the natToolbarItemPosition attribute — DOM, screen-reader and keyboard order always match what you see. The toolbar is a generic host: drop in your own components — a search field, a dropdown — and they join the same roving focus.

Search & filter menu (user components)

Two consumer-built components share one toolbar — neither is shipped by the library. The <app-table-search toolbar> drives the real global filter; the Actions button is a hand-wired disclosure of preset filters (the ARIA menu directive can't share a natToolbarItem). Pick a preset to filter the table — it composes with the free-text search. Both controls join the toolbar's roving focus.
