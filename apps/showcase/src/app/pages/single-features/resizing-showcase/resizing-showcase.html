Column Resizing Feature

Demonstrates interactive column resizing with pointer drag and keyboard support. Every column declares a minimum and maximum size, and the table announces each new width to screen readers. Switch the width mode to compare Fill (columns stretch to fill the container) with Fixed (authoritative pixel widths; the region scrolls horizontally).

How To Resize A Column

Drag: grab the thin handle on a header's edge and drag to resize the column.

Keyboard:

  1. Tab to a column's resize handle.
  2. Press Left Arrow / Right Arrow to resize in small steps.
  3. Hold Shift with an arrow key for larger steps.
  4. Press Home / End to jump to the minimum or maximum width.
