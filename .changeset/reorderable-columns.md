Add opt-in reorderable table columns backed by TanStack state and Angular CDK drag-drop. Leaf headers can now be reordered by drag and keyboard, unpinned column order is preserved in table state, pinned columns reorder within their own left and right regions, and the built-in header action buttons continue to work while reordering is enabled.
